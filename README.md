# portal-chn-pages

## 项目简介

本仓库用于归档和发布多个独立的 HTML 页面，涵盖各类实用工具、信息展示、交互页面等。所有页面均为独立文件，不依赖外部服务器或特定域名，便于本地浏览与离线使用。

## 目录说明

```
portal-chn-pages/
├── pages/          # 存放所有 HTML 页面文件
├── assets/         # 页面所需的静态资源（CSS、JS、图片等）
├── index.html      # 页面索引或导航入口（如有）
├── README.md       # 本文件
└── LICENSE         # 开源许可文件（如适用）
```

- `pages/`：每个 HTML 文件对应一个独立页面，文件名应简洁，反映页面功能。
- `assets/`：按类型或页面组织资源文件，避免重复冗余。
- `index.html`：若仓库需提供总览或导航功能，可在此文件中列出所有页面链接。

## 页面归档说明

- 所有页面均为**静态 HTML**，不含后端依赖。
- 页面内引用的资源（如样式、脚本、图片）应使用相对路径或内联方式，确保在本地或任何静态托管环境中均可正常显示。
- 建议每个页面自包含（self-contained），减少跨文件依赖，便于单独分享或部署。
- 若页面需外部资源（如 CDN 库），请注明来源及版本，并尽量使用常见稳定版本。

## 维护说明

- 欢迎提交新页面或改进现有页面。请确保新页面功能完整、代码整洁，并放置在 `pages/` 目录下。
- 提交前请检查页面是否包含敏感信息、外部链接或可能引起误解的内容。
- 若需更新索引文件（如 `index.html`），请同步修改。
- 本仓库不接受涉及商业推广、广告、恶意脚本或违法内容的页面。
